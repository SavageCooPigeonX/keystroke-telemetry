## THE INTERROGATION ROOM

The headline feature: a dark-themed web interface (port 8236) where the codebase's most problematic modules interrogate the human operator. Each module gets ranked by urgency — escalation level, ignored passes, chronic bug status, fix failures, and cognitive hesitation score. The top 15 suspects get loaded into a lineup.

**The twist:** initial deployment revealed all 8 top modules asked the same question — "Why haven't you fixed this hardcoded import?" — because they all had identical `hardcoded_import` bugs logged against them. The question template system was diversified with 30+ question variants across 10 buckets (identity, intent, relationship, coupling, history) with position-based rotation so no two modules in the same session ask the same thing.

Answers get written to `logs/interrogation_answers.jsonl` and injected into the thought completer's Gemini prompts as "OPERATOR DECISIONS." The loop closes: module asks question → operator answers → thought predictor learns → predictions improve → next interrogation adapts.

## THE ZOMBIE CLEARING

**Root cause analysis** of the broken auto-fixer revealed a fundamental flaw: the bug scanner was flagging patterns inside docstrings and comments as real imports. A regex pattern matching `_seq\d+_v\d+_d\d+__[\w]+` was finding hits in documentation strings, not actual import statements.

**The fix:** AST-based `_non_code_lines()` function added to the hardcoded import scanner. It parses the source with Python's `ast` module, identifies all string literal nodes (docstrings) and comment lines, and excludes them from scanning.

**The result:** 0 actual hardcoded imports exist in the codebase. All 15 escalated modules were phantoms — their bugs had been fixed but the escalation engine's state was frozen since April 6. A `--clear-zombies` CLI command now auto-detects and purges these entries.

**Before:** 15 modules stuck at ACT/VERIFY level, endlessly attempting to fix nonexistent bugs.
**After:** Clean escalation state. Zero modules. The engine can start fresh.
